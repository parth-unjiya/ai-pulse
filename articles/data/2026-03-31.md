2. **Claude Code Source Code Leaked** (The Decoder)

        → The source code of Claude Code, a popular AI coding tool, has been leaked via a map file in their NPM registry. This leak exposes over 500,000 lines of TypeScript code, potentially allowing developers to create custom modifications and improvements. The leak has sparked a discussion on the Hacker News community, with some developers already rebuilding the executable from the leaked sourcemap. This incident highlights the importance of security and intellectual property protection in the AI industry. As a result, we can expect to see more emphasis on security measures and potential legal actions to protect AI technologies.
